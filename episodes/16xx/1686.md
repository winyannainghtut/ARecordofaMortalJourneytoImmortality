# 1686 — Emergence of the Dark Beasts

Dark Beasts များသည် ကျယ်ပြောလှသော ရေခဲပြင်နယ်မြေတွင် တန်ခိုးအရှိဆုံးသားရဲများမဟုတ်သော်လည်း၊ ရှေးသားရဲများပင်လျှင် ညဘက်တွင် ရင့်ကျက်သော Dark Beasts အစုအဝေးကို မရင်ဆိုင်ချင်ကြပေ။
အကြောင်းတစ်ခုခုကြောင့်၊ ၎င်းသည် နေ့၏သုံးပုံနှစ်ပုံခန့်တွင် Dark Beast သစ်တောတွင် ညအချိန်ဖြစ်သဖြင့် ဆန့်ကျင်ဘက်လက်ကျန်နာရီအနည်းငယ်သာ ကျန်ရှိတော့သည်။
တောနက်ထဲတွင် ထောင်ပေါင်းများစွာသော အမှောင်တိရစ္ဆာန်များ နေထိုင်ကြပြီး အများစုမှာ အရွယ်ရောက်ပြီးသော အမှောင်သားရဲများဖြစ်သည်။
ထို့ကြောင့်၊ Dark Beast Forest သည် Myriad Fowl Ridge နှင့် Eight Omens Sea တို့နှင့် တိုက်ရိုက်အနီးနားရှိသော်လည်း၊ အနီးနားရှိ တန်ခိုးကြီးသော ရှေးသားရဲများ မည်သည့် သစ်တောကိုမျှ ကျူးကျော်ရန် ရည်ရွယ်ချက်မရှိပေ။
ဤသစ်တောသည် Dark Beasts ၏ နောက်ဖေးခြံနှင့်တူပြီး ၎င်းအတွင်းနေထိုင်သော အခြားသားရဲအားလုံးနီးပါးသည် အမှောင်သားရဲများ၏ သားကောင်အဖြစ်သာ လုပ်ဆောင်နိုင်ကြသည်။
ညဘက်တွင် ပိုမိုကောင်းမွန်အောင်လုပ်ဆောင်နိုင်သော အခြားသားရဲအမျိုးအစားအချို့ရှိကာ ၎င်းတို့သည် Dark Beasts များကို ခြိမ်းခြောက်မှုအချို့ပြုလုပ်နိုင်သော်လည်း ၎င်းတို့အားလုံးသည် ကိန်းဂဏန်းများအပြင် စွမ်းအားတစ်ခုချင်းစီအရ Dark Beasts များထက် အဆပေါင်းများစွာ ယုတ်ညံ့ပါသည်။
အဲဒီနေ့မှာပဲ Dark Beast သစ်တောရဲ့ နယ်နိမိတ်နဲ့ ကီလိုမီတာ 100 အကွာမှာ အရှည်ပေ 70 မှ 80 ရှိတဲ့ ဧရာမစပါးအုံးမြွေကြီးတစ်ကိုယ်လုံးရှိ အပြာရောင်အကြေးခွံတွေနဲ့ တင်းမာနေတဲ့ ထိပ်တိုက်ရင်ဆိုင်မှုမှာ အမှောင်သားရဲတစ်ကောင်နဲ့ ရင်ဆိုင်ခဲ့ရပါတယ်။
ဤစပါးအုံးမြွေသည် သာမန်စပါးအုံးမြွေနှင့် မတူပုံပေါက်သော်လည်း လက်တွေ့တွင်၊ ၎င်းသည် အမှောင်သားရဲသစ်တောရှိ သားရဲအမျိုးအစားများထဲမှ တစ်မျိုးဖြစ်ပြီး အမှောင်သားရဲများကို စိန်ခေါ်နိုင်သည်။
ဤစပါးအုံးမြွေ အမျိုးအစားများကို Azure Night Python ဟုခေါ်ကြပြီး ညအချိန်တွင် ၎င်းတို့၏ အစွမ်းထက်သော ခန္ဓာကိုယ်များ သိသိသာသာ အားကောင်းလာကာ ည၏ စွမ်းအားကို အားဖြည့်ပေးကာ ပြင်းထန်သော အခြေအနေသို့ ဝင်ရောက်ရန် ညဘက်တွင်လည်း ဆွဲဆောင်နိုင်မည်ဖြစ်သည်။
ရိုင်းစိုင်းသော အသွင်ပြောင်းပြီးနောက်၊ ဤစပါးအုံးမြွေကြီးများသည် ရင့်ကျက်သော အမှောင်သားရဲများနှင့် ယှဉ်နိုင်သော စွမ်းအားများ ပိုင်ဆိုင်ထားပြီး ၎င်းတို့သည် တိုက်ပွဲတွင် အလွန်ရက်စက်ပြီး သွေးဆာနေပါသည်။
သို့သော်၊ ဤအသွင်ပြောင်းမှုမျိုးသည် ဤ Azure Night Python များအတွက် အနုတ်လက္ခဏာဆောင်သော နောက်ဆက်တွဲဆိုးကျိုးများစွာကို သဘာဝအတိုင်းတင်ပြပါသည်။
၎င်းတို့သည် လာမည့်နှစ်အနည်းငယ်အတွင်း ပြင်းထန်စွာ ဆုတ်ယုတ်သွားရုံသာမက ၎င်းတို့၏ သက်တမ်းကိုလည်း လျှော့ချမည်ဖြစ်သည်။ ထို့ကြောင့်၊ ၎င်းတို့အနက်မှ ၎င်းတို့သည် ဆိုးရွားသောအန္တရာယ်ကို မတွေ့မချင်း ထိုကဲ့သို့ အသွင်ကူးပြောင်းမှုကို ကြုံတွေ့ရမည်မဟုတ်ပေ။
ကံမကောင်းစွာပဲ အဆိုပါ Azure Night Python များအတွက် ၎င်းတို့၏ အတွင်းအူတိုင်များသည် Dark Beasts များကို စားသုံးရန်အတွက် အလွန်အကျိုးရှိသောကြောင့် Dark Beast Forest တွင် အချင်းချင်း တွေ့ကြုံရပါက သေခြင်းတရားသည် ချက်ခြင်းဆိုသလို ဖြစ်ပွားလာမည်ဖြစ်သည်။
ဒါက အခုဖြစ်ပျက်နေတာ အတိအကျပဲ။
Azure Night Python သည် ၎င်း၏ ဦးခေါင်းကို မြင့်မားစွာ မြှင့်တင်ထားပြီး ၎င်း၏ နက်မှောင်သော ခရမ်းရောင် ခြည်ခြည်မျှင်လျှာကို ခြိမ်းခြောက်သည့်ပုံစံဖြင့် ပွတ်တိုက်နေပါသည်။ ၎င်း၏ခန္ဓာကိုယ်ပေါ်ရှိ ပြာမှုန်မွှားမွှားအကြေးခွံများအားလုံးသည်လည်း အဆုံးတွင် မတ်တတ်ရပ်နေပြီး၊ ၎င်းသည် ခဏတာအတွင်း ခုန်ပေါက်ရန် အဆင်သင့်ဖြစ်ပုံရသည်။
၎င်းကို မျက်နှာမူထားသည့် Dark Beast သည် ၎င်း၏သာမန်အရွယ်အစားထက် သုံးလေးဆခန့် ဖောင်းနေသော ဝံပုလွေတစ်ကောင်၏အသွင်အပြင်ကို မွေးဖွားခဲ့သည်။ ၎င်းသည် လုံးဝ မှင်သက်သော အနက်ရောင်ဖြစ်ပြီး ၎င်း၏အမြီးသည် ကျားသစ်ကဲ့သို့ ရှည်လျားပြီး ပါးလွှာကာ ချောမောပြီး ကျက်သရေရှိသော ကြောင်ရုပ်အသွင်ကို ပေးသည်။
ဒီ Dark Beast ဟာ သာမာန် ရင့်ကျက်တဲ့ Dark Beast ထက် အနည်းငယ် ပါးလွှာပြီး မကြာသေးမီကမှ ရင့်ကျက်မှု ရောက်ရှိလာတာ ထင်ရှားပါတယ်။ သို့သော်၊ Azure Night Python တွင် ၎င်း၏မျက်လုံးများတွင် ရက်စက်စွာ တောက်ပြောင်နေသည့် တောက်ပမှုနှင့်အတူ Azure Night Python တွင် တောက်ပနေသောကြောင့် အနည်းငယ်မျှ ကျောမထောက်ထားပေ။
ထူးဆန်းသည်မှာ၊ Dark Beast သည် ဧရာမစပါးအုံးမြွေကြီးကိုသာ စိုက်ကြည့်နေပြီး ချက်ချင်းပင် ၎င်းဆီသို့ မခုန်တက်ပေ။
ရင့်ကျက်သော Dark Beats နှင့် berserk Azure Night Python များအကြား တိုက်ပွဲအများစုသည် နောက်ဆုံး၏ အတွင်းအူတိုင်ကို လုံခြုံအောင်ထားခြင်းဖြင့် ပြီးဆုံးသွားသော်လည်း Azure Night Python များမှ Dark Beasts များကို ကိုက်စားသည့် ကိစ္စများလည်း ရှိပါသည်။
ထို့ကြောင့်၊ Dark Beast သည် ဤနေရာတွင် Azure Night Python ကိုတွေ့လိုက်ရသည့်အတွက် ဝမ်းသာမိသော်လည်း ချက်ခြင်းတိုက်ပွဲသို့ အလျင်စလိုမဖြစ်ပါ။
ရိုင်းစိုင်းသော Azure Night Python သည် ပိုမိုအားကောင်းလာမည်မှာ သိသာထင်ရှားသော်လည်း ၎င်းသည် ၎င်း၏အသွင်ပြောင်းသွားချိန်တွင် ၎င်း၏အားနည်းချက်ကို ဖော်ထုတ်နိုင်မည်ဖြစ်ပြီး Dark Beast သည် အဆိုပါအားနည်းချက်ကို စက္ကန့်ပိုင်းအတွင်း အခွင့်ကောင်းယူနိုင်ပါက၊ ၎င်းသည် ၎င်း၏ပြိုင်ဘက်ကို အချိန်ကိုက်တစ်ချက်တည်းဖြင့် သတ်ပစ်နိုင်သည်။
တစ်ချိန်တည်းတွင်၊ Azure Night Python သည် အခိုက်အတန့်တွင် ရူးသွပ်သွားလိမ့်မည်ဟု ညွှန်ပြနေသည့် ဟန်ပန်ကို လက်ခံကျင့်သုံးခဲ့သော်လည်း လက်တွေ့တွင်မူ ၎င်းသည် မည်သည့်အရာကိုမျှ တုန်လှုပ်ချောက်ချားရန် မဝံ့ရဲခဲ့ပေ။
ထို့ကြောင့် တင်းမာသော ပဋိပက္ခများ ဆက်လက်ဖြစ်ပွားခဲ့သည်။
Dark Beast သစ်တောရှိ သားရဲများအားလုံးသည် နေ့ဘက်တွင် ၎င်းတို့၏ လိုဏ်ဂူများတွင် အနားယူနေကြသည့် ညနေသတ္တဝါများဖြစ်သည်။

ဤသစ်ပင်များသည် အလွန်မြင့်မားပြီး သစ်တောအစွန်းနှင့် အနည်းငယ်အလှမ်းဝေးသော်လည်း အထက်တွင်ရှိသော စိမ်းလန်းစိုပြေသော အကိုင်းအခက်များနှင့် အရွက်များသည် ကောင်းကင်တစ်ခုလုံးကို ဖုံးကွယ်လုမတတ်လောက်အောင် ထူထပ်နေသေးသည်။ အကယ်၍ တစ်စုံတစ်ယောက်သည် ဤစိမ်းလန်းစိုပြေသော မြက်ခင်းပြင်များကို စမ်းကြည့်မည်ဆိုလျှင်၊ ကျယ်ပြောလှသော အမှောင်ထု၏ မြင်ကွင်းကြောင့်သာ ၎င်းတို့ကို ကြိုဆိုနိုင်ပေလိမ့်မည်။
လက်ရှိအချိန်သည် ည၏အမှောင်ဆုံးအချိန်ဖြစ်ပြီး ၎င်းသည် အနီးနားရှိ တောက်ပသောအပင်များမဟုတ်ပါက၊ ဤနေရာသည် လုံးဝအနက်ရောင်ဖြစ်နေပေလိမ့်မည်။
အချိန်တွေ တဖြည်း ဖြည်း ကုန်လွန်လာပြီး ထိပ်တိုက်တွေ့မှုတွေ ဆက်ဖြစ်နေပါတယ်။ ဤအချိန်၌ သတ္တဝါ နှစ်ဦးစလုံးသည် အတော့်ကို စိတ်ပျက်သွားကြသည်။
နောက်ဆုံးတော့ သည်းမခံနိုင်တော့တဲ့ အမှောင်သားရဲကြီးပါပဲ။

အဲဒီနောက် ချက်ချင်းဆိုသလိုပဲ Dark Beast ရဲ့ အပေါ်က လေထဲမှာ ပြာပြာရောင် အလင်းတန်းတစ်ခု ထွက်ပေါ်လာပြီး စပါးအုံးမြွေကြီး ဦးခေါင်းဟာ အလင်းရောင်အတွင်းကနေ ထွက်ပေါ်လာပါတယ်။ ထို့နောက် စပါးအုံးမြွေ၏ဦးခေါင်းသည် အံ့အားသင့်ဖွယ်အရှိန်ဖြင့် ပြိုကျလာကာ Dark Beast ခန့်မှန်းချက်တစ်ခုအား မျိုချလိုက်ပြီး အခြားတစ်ကောင်ဆီသို့ အလျင်အမြန်သွားခဲ့သည်။
ဤအခိုက်အတန့်တွင် တကယ့်အမှောင်သားရဲကြီးသည် ဘေးမှ ထွက်ပေါ်လာပြီး စပါးအုံးမြွေကြီးဆီသို့ မထိမီ ဓားမြှောင်နှင့်တူသော သွားတန်းများ ထွက်ပေါ်လာပြီး ၎င်း၏ပါးစပ်ကို ကျယ်ကျယ်ကျယ်ကျယ်ဖွင့်ကာ စပါးအုံးမြွေကြီးဆီသို့ လှမ်းကန်လိုက်သည်။
Azure Night Python ၏ကိုယ်ထည်သည် မည်မျှထူနေသော်လည်း ထိုမေးရိုးကြီးများကို ဖိထားလိုက်မည်ဆိုလျှင် ၎င်းကို နှစ်ခြမ်းကွဲသွားလိမ့်မည်မှာ သေချာပါသည်။
သို့သော်လည်း ရူးသွပ်သွားပြီးနောက်၊ Azure Night Python ၏အမြန်နှုန်းသည် Dark Beast ထက်မနိမ့်ဘဲ၊ ၎င်းသည် ၎င်း၏ပါးစပ်ပေါက်ကိုမဖွင့်မီ ခေါင်းကိုရုတ်တရက်ဆုတ်ခွာကာ Dark Beast ဘက်သို့လှည့်သွားခဲ့သည်။
ဤအခိုက်အတန့်တွင်၊ ၎င်း၏မူလပစ်မှတ်ဖြစ်သော အခြား Dark Beast ၏ကိုယ်ထည်သည် Azure Night Python ဆီသို့ တိုက်ရိုက်ထိသွားသဖြင့် မယုံနိုင်လောက်အောင် ကျယ်ပြန့်သွားခဲ့သည်။
ထို့ကြောင့် သားရဲနှစ်ကောင်သည် ကျယ်လောင်စွာ ပေါက်ကွဲသံများကြားတွင် ခိုက်မိသွားကာ ၎င်းတို့သည် အတော်လေး လိုက်ဖက်ညီပုံရသည်။
သူတို့နှစ်ဦးကို မသိလိုက်ဘဲ၊ ပေ ၂၀ ပတ်လည် အချင်းဝက်ရှိ ကြီးမားသော အပေါက်တစ်ခုကို ၎င်းတို့နှင့် ၁၀ ကီလိုမီတာအကွာတွင်ရှိသော သာမန်ပုံစံ ဧရာမသစ်ပင်ကြီးအဖြစ် ထွင်းထုထားသည်။
သစ်ပင်တွင်းရှိ အပေါက်အတွင်း၌ တင်ပျဉ်ခွေပုံစံဖြင့် လူသုံးယောက်ထိုင်လျက် လေလယ်တွင် ပျံဝဲနေသော မှန်တစ်ချပ်ကို အကဲဖြတ်ကာ သူတို့၏မျက်နှာများကို စူးစူးစိုက်စိုက်ကြည့်နေကြသည်။
မှန်၏နောက်ဘက်သည် ငွေရောင်အလင်းတန်းများဖြင့် တောက်ပနေပြီး၊ မျက်နှာပြင်၏မှန်သည် အလွန်ကြည်လင်ပြတ်သားသော ပုံရိပ်ကို ပြသနေပါသည်။ ပုံသည် Dark Beast နှင့် Azure Night Python အကြား တိုက်ပွဲမှလွဲ၍ အခြားပုံမဟုတ်ပေ။
အကွာအဝေးရှိ မြင်ကွင်းအားလုံးကို မှန်ထဲတွင် အပြည့်အဝ ပုံတူကူးထားပြီး၊ အရာအားလုံးသည် ၎င်းတို့၏ တကယ့်အရွယ်အစားထက် အဆမရေမတွက်နိုင်သော သေးငယ်သည်မှတပါး၊
ဒီသုံးယောက်ထဲမှာ တစ်ယောက်က အပြာရောင်ဝတ်ထားတဲ့ အမျိုးသား၊ တစ်ယောက်က သူ့ခေါင်းထက်မှာ အဖြူထည်ဝတ်ထားတဲ့ အမျိုးသမီးတစ်ယောက်ဖြစ်ပြီး နောက်ဆုံးကတော့ ပိန်ပိန်ပါးပါး အသားဝါတဲ့ အမျိုးသားတစ်ယောက်ပါ။
၎င်းတို့သည် ဟန်လီ၊ လျူရှုအာ နှင့် ရှီကွမ်းတို့ဖြစ်ပြီး တစ်လနီးပါးအကြာတွင် ဤနေရာသို့ နောက်ဆုံးရောက်ရှိခဲ့သော ဟန်လီ၊
သူတို့ သုံးယောက်စလုံး မှိတ်တုတ်မှိတ်တုတ် မမှိတ်မသုန် မျက်လုံးများဖြင့် မှန်ကို စိုက်ကြည့်ရင်း ကြမ်းတမ်းသော အမူအရာများ ဝတ်ဆင်ထားကြသည်။
တစ်နာရီနီးပါးကြာပြီးနောက်၊ ပုံထဲက Dark Beast သည် ရာနှင့်ချီသော လင်းနို့နက်များအဖြစ် ကွဲသွားခဲ့ပြီး နောက်ဆုံးတွင် ဒဏ်ရာရှိသော Azure Night Python ကို သေဆုံးသည်အထိ ၀တ်ဆင်ကာ စိတ်ဓာတ်ပြင်းထန်စွာဖြင့် ထွက်မသွားမီ ၎င်း၏သားကောင်၏အူတိုင်ကို ကိုက်စားခဲ့သည်။
Liu Shui'er သည် မှန်ပေါ်သို့ ဂါထာတံဆိပ်ကို ချလိုက်ပြီးနောက် သူမသက်ပြင်းချရင်း ပုံရိပ်ပျောက်ကွယ်သွားကာ "ဒီအမှောင်သားရဲတွေကို မင်းကိုယ်တိုင်မြင်တွေ့ပြီးနောက် မင်းဘယ်လိုထင်လဲ"
